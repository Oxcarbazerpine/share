# share

用 GitHub Pages 托管 HTML 文件，方便分享链接给他人。

## 使用方法

1. 将 HTML 文件放到仓库根目录（或子目录）。
2. 推送到 `main` 分支后，GitHub Actions 会自动部署到 GitHub Pages。
3. 访问 `https://oxcarbazerpine.github.io/share/` 即可看到页面，也可直接分享该链接。

> **首次使用**：请在仓库的 **Settings → Pages** 中将 Source 设置为 **GitHub Actions**，然后工作流就会自动接管部署。

Host HTML files via GitHub Pages and share links easily.

3. Share `https://oxcarbazerpine.github.io/share/` (or any sub-path) with others.
